# MERN Todo List

A simple Todo List application built with the MERN stack (MongoDB, Express, React, Node.js).

## Features

- Add, view, update and delete todos
- Modern React frontend (Vite)
- Express/Node.js backend
- MongoDB for data storage

## Getting Started

### Prerequisites

- Node.js
- MongoDB

### Installation

#### Server

```
cd server
npm install
```

#### Client

```
cd client
npm install
```

### Running the App

1. Start MongoDB.
2. Start the server:
   ```
   cd server
   npm start
   ```
3. Start the client:
   ```
   cd client
   npm run dev
   ```

## Folder Structure

- `client/` - React frontend
- `server/` - Express backend
